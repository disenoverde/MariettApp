## 🚀 Cómo usar:

### 1. Configurar Firebase (5 minutos)
```bash
# Sigue: SETUP-FIREBASE.md
```

### 2. Instalar
```bash
npm install
cp .env.example .env
# Edita .env con tus credenciales Firebase
```

### 3. Ejecutar
```bash
npm run dev
```

### 4. Probar
1. Login con tu usuario Firebase
2. Crea un cliente
3. Abre su ficha
4. Completa Historia de Salud ✅
5. Define Objetivos ✅
6. Crea Sesiones ✅
7. Genera PDFs ✅
